# 简介

Netty 是一个 asynchronous event-driven（异步事件驱动）的网络应用框架，是一个用于快速开发高性能、可扩展协议的服务器和客户端。

换句话说，Netty 是一个 NIO 客户端服务器框架，使用它可以快速简单地开发网络应用程序，比如服务器和客户端的协议。Netty 大大简化了网络程序的开发过程比如 TCP 和 UDP 的 socket 服务的开发。
